Ontology Specification Draft

Abstract

The COGITO Platfrom ontology aims at modelling the entities considered in the the Digital Twin Platform, a component of the COGITO architecture

Introduction back to ToC

This document describes the results of the Platform ontology for the COGITO project.

Construction Twin Thingc back to ToC or Class ToC

IRI: https://cogito.iot.linkeddata.es/platform#PlatformThing

A subclass of a td:Thing and, as such, an abstraction of a physical or a virtual entity whose metadata and interfaces are described by a WoT Thing Description. In this case, an entity of a construction twin, namely, a facility:Project, a facility:Element, a qual:Image. a qual:PointCloud and a qual:Mesh.
has super-classes
Thing c
has sub-classes
Digital Twin c, Information Resource c
