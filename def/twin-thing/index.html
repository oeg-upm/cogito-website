Ontology Specification Draft

Abstract

The COGITO Construction Twin Thing ontology aims at modelling the (WoT) things considered in a construction twin.

Introduction back to ToC

This document describes the results of the Construction Twin Thing ontology for the COGITO project.

Construction Twin Thingc back to ToC or Class ToC

IRI: https://cogito.iot.linkeddata.es/constwinthing#ConstructionTwinThing

A subclass of a td:Thing and, as such, an abstraction of a physical or a virtual entity whose metadata and interfaces are described by a WoT Thing Description. In this case, an entity of a construction twin, namely, a facility:Project, a facility:Element, a qual:Image. a qual:PointCloud and a qual:Mesh.
has super-classes
Thing c
has sub-classes
Element c, Image c, Mesh c, Point Cloud c, Project c
